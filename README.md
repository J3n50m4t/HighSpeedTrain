# HighSpeedTrain

Adds 3 faster Trains and a cargo wagon.
* Mk1: 4 Times faster  @ 73% effectivity
* Mk2: 5 Times faster  @ 63% effectivity
* Mk3: 7 Times faster  @ 50% effectivity
* Wagon: Inventory Size 60 and Maxspeed
